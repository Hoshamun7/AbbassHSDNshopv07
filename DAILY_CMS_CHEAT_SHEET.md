## üìù Quick Content Creation

### Adding a New Product
1. Go to **Products** ‚Üí **New Product**
2. Fill required fields:
   - Product ID (e.g., `jalabiya-red-2024`)
   - Title (English & Arabic)
   - Price range
   - Category
3. Upload product image
4. Add description in both languages
5. Save as **Draft** or **Publish**

### Updating Statistics
1. Go to **Statistics**
2. Find the statistic to update
3. Click to edit
4. Update the number/text
5. Save changes

### Creating Blog Post
1. Go to **Blog Posts** ‚Üí **New Post**
2. Add title in both languages
3. Write content in markdown editor
4. Add featured image
5. Set publication date
6. Add tags for searchability

## üîß Common Fixes

### Content Not Appearing
1. Check if published (not draft)
2. Clear browser cache
3. Wait 2-3 minutes for CDN
4. Try hard refresh (Ctrl+F5)

### Images Not Loading
1. Check file size (max 10MB)
2. Verify file extension
3. Re-upload if corrupted
4. Check internet connection

### Login Issues
1. Clear browser cookies
2. Try incognito mode
3. Reset password if needed
4. Contact support if persistent
